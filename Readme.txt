this app is based on real application which should be used in Eissmann company. Data structure is the
same, but data itself are changed. 
    1. open vos_demo in terminal
        right-click on vos_demo -> open in integrated terminal
    2. run 'npm start' command 
        npm start executes script defined in package.json and starts dev server which
        serves the SAPUI5 application and provides a live preview of the app in browser.

    3. in the app there is a expandable footer, with two buttons. Use this to start and stop
        the demo. Of course in the 'real' application there is no footer with start and stop.
        It should be it's designed to show shipments within one week at start-up with ascending 
        order from time to load.
